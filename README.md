# FiuFiu 🐥 - Em memória do Twitter

Aplicação desenvolvida para a Prova Prática de Desenvolvimento de Software para Web (G2).
Uma rede social simples no estilo do Twitter antigo, contendo sistema de autenticação, postagens, curtidas e uma Feature Extra de alternância de tema (Dark/Light mode). O design foi aprimorado para refletir fielmente a interface clássica, incluindo avatares dinâmicos.

## 🚀 Tecnologias Utilizadas
- **Front-end:** React
- **Back-end:** Node.js com Express
- **Banco de Dados:** SQLite

## ⚙️ Como rodar a aplicação localmente

### 1. Pré-requisitos
Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

### 2. Rodando o Back-end (Servidor e Banco de Dados)
Abra um terminal na pasta raiz do projeto e execute:
```bash
cd Backend
npm install
node server.js
